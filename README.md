# 🎉 Event Management System

A web-based Event Management System developed using PHP and MySQL that allows users to discover events, book tickets, and manage bookings.

## 🚀 Features

- User Registration and Login
- Event Discovery
- Ticket Booking
- Organizer Dashboard
- Admin Panel
- QR Payment Upload
- Ticket Generation
- Password Recovery
- Responsive Design

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## 📂 Project Structure

```
admin/
assets/
includes/
organizer/
uploads/
index.php
dashboard.php
discover.php
event.php
```

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/vijay23ai/event-management.git
```

2. Import the database into MySQL.

3. Configure database settings in:

```text
includes/db.php
```

4. Start Apache and MySQL using XAMPP.

5. Open:

```text
http://localhost/event-management
```
